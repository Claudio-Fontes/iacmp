# Contribuindo

O guia completo de contribuição vive em [docs/contribuindo.md](docs/contribuindo.md).

Resumo rápido:

1. `npm install`
2. `npm run build`
3. `npm test`
4. Abra um PR descrevendo a mudança em pt-BR.

Convenções de commit, scripts disponíveis, como adicionar comandos, constructs
e providers customizados estão todos detalhados em `docs/contribuindo.md`.
